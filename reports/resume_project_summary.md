# Resume Project Summary

## Windows Detection Engineering Lab

Designed and implemented a Windows Detection Engineering lab using Kali Linux and Windows 11 to simulate and investigate Windows security events.

Configured Sysmon to generate detailed telemetry and developed Sigma detection rules for PowerShell abuse, process creation, network connections, file creation, registry modifications, and credential dumping techniques.

Performed SOC-style incident investigations by correlating Sysmon events, analyzing attack chains, classifying incident severity, and documenting findings using professional investigation reports.
