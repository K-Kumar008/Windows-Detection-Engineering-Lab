# LinkedIn Project

Recently completed my Windows Detection Engineering Lab.

The project was built using Kali Linux and a Windows 11 virtual machine.

Key activities included:

• Installing and configuring Sysmon
• Collecting Windows telemetry
• Analyzing Event IDs 1, 3, 11, and 13
• Developing Sigma detection rules
• Investigating attack chains
• Mapping detections to the MITRE ATT&CK framework
• Writing professional incident reports

This project significantly improved my understanding of Windows logging, detection engineering, and SOC investigation workflows.
