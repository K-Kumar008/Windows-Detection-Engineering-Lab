# SOC Interview Notes

## Explain Sysmon

Sysmon is a Windows system monitoring tool from Microsoft Sysinternals.

It records detailed telemetry such as:

- Process creation
- Network connections
- File creation
- Registry modifications

---

## Explain Sigma

Sigma is a platform-independent detection rule language.

Sigma rules can be converted into SIEM-specific queries.

---

## Why Sysmon?

Windows Event Logs are limited.

Sysmon provides much richer telemetry.

---

## Why Sigma?

Write once.

Use across multiple SIEM platforms.
